# Site LVPSA

Site web React/Vite prêt à déployer sur Vercel.

## Déploiement Vercel
1. Aller sur https://vercel.com
2. Se connecter avec Google
3. Cliquer sur Add New > Project
4. Importer ce dossier ou le déposer dans Vercel
5. Framework preset : Vite
6. Build command : npm run build
7. Output directory : dist
8. Cliquer Deploy

## Modifier le site
- Texte principal : `src/main.jsx`
- Logo : `public/logo.jpg`
- Courriel : chercher `Liguevpsa@gmail.com` dans `src/main.jsx`
- Lien météo : chercher `weatherLink`
- Lien d'inscription : chercher `registrationLink`
